# Инструкция по добавлению поля image_urls

## Проблема
Сейчас сохраняется только одно главное изображение, остальные теряются при сохранении в БД.

## Решение
Нужно добавить поле `image_urls` в таблицу `properties` в Supabase.

## Шаги:

1. Откройте Supabase Dashboard: https://app.supabase.com
2. Выберите ваш проект
3. Перейдите в **SQL Editor** (в левом меню)
4. Нажмите **New Query**
5. Скопируйте и выполните следующий SQL:

```sql
ALTER TABLE properties 
ADD COLUMN IF NOT EXISTS image_urls TEXT;

COMMENT ON COLUMN properties.image_urls IS 'JSON массив URL изображений для объекта недвижимости';
```

6. Нажмите **Run** (или Ctrl+Enter)
7. Должно появиться сообщение об успешном выполнении

## Проверка
После выполнения миграции:
- Попробуйте сохранить объект с несколькими изображениями
- Проверьте консоль сервера - должно быть видно "Saving image_urls: X images"
- При загрузке объектов все изображения должны отображаться

## Альтернатива (если нет доступа к Supabase)
Если у вас нет доступа к Supabase Dashboard, можно использовать Supabase CLI:
```bash
supabase db push
```
Или выполнить SQL через psql, если есть доступ к базе данных напрямую.
